#　回答例 
なぜ php artisan key:generate は重要なの？
php artisan key:generate は、Laravel アプリがセキュアに動作するために アプリケーションキー（APP_KEY） を生成するコマンドです。
🔍 APP_KEY が果たす主要な役割
① 暗号化の安全性を担保する
Laravel の以下の処理で 暗号化 が使われています：
パスワードリセットトークン
暗号化された Cookie（特に EncryptCookies ミドルウェア）

Crypt::encrypt() / decrypt()

APP_KEY が未設定だと暗号化できず、正しく復号もできません。
APP_KEY は アプリ固有の秘密鍵 なので、各環境で必ず生成し、GitHub などに公開しないよう .env に保存します。

# MVC の流れコメント（模範解答）
```
 * 【Laravel の MVC の流れ】
 * ブラウザからURLにアクセスすると、まずルートが呼び出される。
 * ルートは、どのコントローラ（または処理）を実行するかを決める。
 * コントローラは必要なデータをモデル（M）から取得し、ビュー（V）へ渡す。
 * ビューは受け取ったデータをもとにHTMLを作成し、レスポンスとしてブラウザへ返す。
 * ルート／コントローラがC、ビューがV、データ処理を行うモデルがMの役割となる。
```
